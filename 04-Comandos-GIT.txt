# GIT REMOTE
despues de crear el repo en github ponemos dentro de la ruta del proyecto:
    git remote add origin https://github.com/josef-666/hello-git.git 
que empareja el proyecto local hacia github


subimos los cambios a github con
    git remote add origin https://github.com/josef-666/hello-git.git

despues de subir todo, modificamos y creamos archivos, los añadimos con git add 
y luego con git commit -m "msj"

git fetch, se descarga el historial sin los cambios

git config pull.rabase false: primero se usa este comando para traer los cambios del repo remoto
                                al local

git pull origin main: se descargar el historial y tambien los cambios

git push: sube los cambios al repo en github
